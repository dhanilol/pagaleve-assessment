## About this project

This project uses Laravel 8 as an assessment for <a href="https://pagaleve.com.br/">Pagaleve</a>.

Laravel is accessible, powerful, and provides tools required for large, robust applications.


## How to start it

-  Set the API configuration (provided by company) in the `.env` file:
    -  PAGALEVE_BASE_URL;
    -  PAGALEVE_API_USER;
    -  PAGALEVE_API_PASSWORD;
 - Run `composer install` following by `npm install` to install all dependencies;
 - Run `php artisan serve` to run a local application serve
